# 项目简介

## 核心功能
- 提供完整的船只检测系统源码，基于YOLOv8模型
- 包含5000张标注数据集，涵盖['boat']单类目标
- 支持图片、视频及摄像头三种输入方式进行检测
- 提供交互式UI界面，可实时显示目标位置、总数和置信度
- 支持批量检测并查看所有结果
- 可自定义检测参数，如置信度阈值和IOU阈值
- 提供模型选择和检测结果保存功能

## 适用场景
- 船只监控与识别
- 海域安全管理
- 水上交通监测
- 海洋科研数据分析

## 技术特点
- 基于YOLOv8目标检测算法，具有较高准确率
- 采用PyQt5构建用户界面，操作友好
- 支持多种输入源，适应不同应用场景
- 提供完整的开发环境和部署文档
- 模型已预训练，可直接使用
